## The AI Engineer Path

Issued: July 8, 2026

A 114-hour, 258-lesson learning path focused on building practical AI-powered applications. This path covers embeddings, semantic search, retrieval-augmented generation (RAG), vector databases, and integrating large language models into web applications.

## Learn React

Issued: March 24, 2026

A 15.1-hour, 357-lesson course covering modern React fundamentals and application development, including components, state, hooks, and building interactive user interfaces.

## Learn AI Agents

Issued: July 8, 2026

A 31-lesson course focused on creating AI agents and understanding agent-based workflows, including tool calling, orchestration, and building applications with the OpenAI Agents SDK.

## Learn Context Engineering

Issued: July 8, 2026

A 15-lesson course covering the design and management of effective context for AI systems, including prompt structure, retrieval strategies, and optimizing what information is passed to language models.
