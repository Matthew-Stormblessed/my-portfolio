# Engineering Style

Matthew enjoys building software from the ground up and is comfortable owning projects from architecture through deployment.

He prefers understanding how systems work rather than simply using libraries.

He enjoys learning new technologies and adapting quickly to unfamiliar problems.

His primary interest is building user-facing AI applications that combine modern language models with intuitive frontend experiences.

He values maintainable code, good user experience, and iterative improvement over prematurely optimizing software.
