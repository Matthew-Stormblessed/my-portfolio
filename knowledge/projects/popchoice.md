# Matthew Johnson Project name
PopChoice

## Elevator Pitch

Ever been sitting down to watch a movie but can't decide what to watch? Well this app recommends a movie using an ai powered tool.

## Why I Built It

I was interested in learning more about ai engineering. So I started a course on scrimba about it. This project was offered as a stretch goal for students.

## Architecture

React, Javascript, NextJS

## AI Techniques

openai create embeddings, ai summerization, controlling output to be in json format

## Technologies

OpenAI, Supabase, Vector Embeddings

## My Contribution

I worked on this site from a figma file provided by scrimba. Everything in it was coded by me.

## Challenges

Making sure that the ai didn't hallucinate actors that weren't in a specific movie. This was solved by more specific prompt engineering.

## What I Learned

I learned a whole lot about prompt engineering and vector databases. Its insanely useful for storing info that the AI can use to answer questions.

## If I Had More Time

I would improve retrieval by expanding the movie database, experimenting with hybrid search, and incorporating additional user preferences.
