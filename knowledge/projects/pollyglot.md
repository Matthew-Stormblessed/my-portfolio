# Matthew Johnson Project name
Pollyglot

## Overview

Pollyglot is an AI-powered translation application that provides conversational translations between English, French, Spanish, and Japanese.

## Why I Built It

I wanted to explore building a conversational AI application while creating an interface that felt natural and easy to use.

## Architecture

Users enter messages into a chat interface built with React. Messages are sent to an Express backend, which calls the OpenAI Chat Completions API to generate translations. Responses are sent back into the conversation, allowing users to continue chatting naturally.

## AI Techniques

- Large Language Models
- Prompt Engineering
- Conversational AI

## Technologies

- React
- Express
- Node.js
- OpenAI Chat Completions

## My Contribution

I built the complete application from Figma designs, including the frontend, backend, AI integration, and deployment.

## Challenges

The primary challenge was creating a smooth chat experience. I implemented automatic scrolling and conversation behavior that makes the interface feel more natural while keeping previous translations easy to reference.

## What I Learned

This project helped me better understand conversational AI interfaces and the importance of creating a user experience that feels responsive and intuitive.
